## 功能模块

### 模型市场

在模型市场，你可以选择当前系统已有的AI模型。每个模型都具备标题、描述、图标、价格等标签。

AI模型包括但不仅限于：

+ 智能增氧控制模型

+ 智能投料控制模型

+ 智能安防模型

+ 智能光照模型

+ 智能排污模型

+ 智能光照模型

### 模型应用

在模型市场选中某个模型后，跳转到模型应用界面。

模型应用界面可以看到模型的标题，

用户在该界面需要选择模型需要的输入和输出。
输入一般是监测设备，如智能排污模型需要选择的设备有：

+ 水质检测设备

+ 鱼情分析摄像头

输出一般是控制设备或者是通知系统，如智能排污模型需要选择输出有：

+ 微滤机设备

另外用户还需要选择模型运行的生效时间段，如

+ 每天

+ 某一时间段

+ 一周的某几天
