## 鱼种采购

鱼种采购有两个核心步骤。首先需要联系鱼种公司，与公司进行洽谈，确定鱼种的数量、品种、价格、规格、起塘条件等记录下来。
当洽谈完多个鱼种公司后，再进入第二个步骤：选择合适的鱼种公司进行采购。

### 洽谈

洽谈记录需要填写：

+ 鱼种公司（从已有记录中选择或者新建）
+ 鱼种品种
+ 鱼种价格
+ 鱼种数量（单位：尾）
+ 起塘条件（是否停食、是否拉练、是否代运）
+ 洽谈时间
+ 计量方式（抽数法、打样法、机器计量、其他）

洽谈记录就像是购物车，洽谈完后，我们在购物车中选择合适的洽谈记录进行采购。

### 开始采购

采购记录有以下节点：

+ 现场准备
  - 鱼种处理（驱虫图片/视频、杀菌图片/视频、备注、断食情况），注意图片和视频都可以上传多个
  - 车辆消毒（图片/视频、备注），注意图片和视频都可以上传多个
  - 鱼种上车（图片/视频、备注），注意图片和视频都可以上传多个

+ 物流车辆
  物流车辆可以选择已有的车辆，也可以新建车辆，新建车辆需要填写
  - 车牌号
  - 公司名称
  - 司机姓名
  - 公司电话
  - 司机电话
  - 备注
  另外，当鱼种公司提供车辆时，可以直接选择物流公司提供即可。

+ 鱼种运输
  运输时需要对鱼种进行抽查是否正常，可以通过上传图片和视频记录，并填写备注。
  抽查可能是多条抽查记录，也可以不抽查。

+ 鱼种下车
  记录鱼种下车的信息，包括：
  - 车辆到达照片/视频
  - 下车照片/视频
  - 备注
  - 下车工人的选择

+ 入池
  入池流程主要是记录鱼种是如何入池的、入池是否正常
  - 入池照片/视频
  - 鱼种正常率（百分比）
  - 备注
  - 入池工人的选择


+ 综合评价
  当鱼种已经正常进食后，采购流程就结束了，此时可以进行综合评价
  - 鱼种质量评分、备注
  - 司机评分、备注
  - 车辆评分、备注
  - 工人的评分、备注
